Linkage Analysis

Linkage Analysis

  • Gene mapping method popular before development of high throughput genotyping techniques
  • 100’s of markers interrogated (causal markers most likely missed)
  • Important genes were found with this approach:
    • CFTR: Cystic Fibrosis
    • HTT: Huntington disease
    • NOD2: Inflammatory bowel disease
  • Markers co-transmitted (recombination rate < 0.5) with the disease susceptibility locus (DSL) help locate the DSL
  • Uses families with many cases (“loaded pedigrees”)

Before the Human Genome Project, disease genes were discovered using linkage analysis. The technology of the time allowed measuring variation in about 100 loci across the genome. Many important genes were discovered with this approach. Linkage analysis is based on co-transmission of genetic markers and disease genes from parent to child. Advantages: a few hundred markers could cover the whole genome. Downside: low resolution and recruiting large families is harder than a large unrelated cohort.

High Penetrance Cancer Susceptibility Genes

  • Genetic linkage (1980s–1990s) found:
    • Breast and ovarian cancers: BRCA1, BRCA2
    • Colorectal cancer: APC, MLH1, MSH2
    • Melanoma: CDKN2A
  • To date, >70 cancer susceptibility genes (CSGs) with relative risks (RR) of 5–100

Sud et al 2017 NRC

Many high penetrance (large effect) cancer susceptibility genes were found using linkage analysis.

By the mid-1990s, discoveries using linkage were dwindling, prompting Risch and Merikangas to propose genome-wide association studies — genotyping ~100K loci — considered science fiction at the time. This added urgency to completing the Human Genome Project.

The Human Genome Project

The first draft of the human genome was announced by President Bill Clinton, who compared it to the moon landing. Technological advances from the HGP made GWAS possible.

Gene Mapping Methods

Linkage analysis

  • Popular before high throughput technology
  • Based on co-transmission of genetic markers and disease genes
  • A few hundred markers can cover the whole genome
  • Low resolution
  • Pedigree/family based

Association mapping

  • Need a large number of markers (> 1 million)
  • Genetic markers in LD (Linkage Disequilibrium) with disease genes
  • Higher resolution
  • Families not needed (easier to recruit large cohorts)

Association methods are based on LD between genetic markers and disease genes. For common variants (MAF > 5%), about 1 million SNPs can tag most common variants. So even if the causal variant isn’t measured but is common, a correlated (in LD) SNP can be detected — called tag SNPs or index SNPs.

Genetic Architecture of Cancer Risk

Expected to Find High Penetrance Genes

Before GWAS, based on what we knew from linkage studies, we expected to find high penetrance (large effect size) genes driving cancer risk.

Found Many Low Penetrance Variants

Instead, GWAS found a large number of variants with modest effect sizes (low penetrance). The genetic architecture of cancer risk turned out to be highly polygenic, requiring better statistical methods to make sense of the discoveries.

Review: Hypothesis Testing

Review: Hypothesis Testing

  • H₀: null hypothesis (e.g., no difference in case-control allele frequencies)
  • Hₐ: alternative hypothesis (there is a difference, correlated with causal variant)
  • Test statistic Z: in many situations associated to a model
  • Significance level: α = allowed type I error (reject null when null is true)
  • p-value = P(observed test statistic more extreme than threshold | H₀)

We reject H₀ if the observed test statistic is more extreme than what we would expect by chance

We reject H₀ if the observed test statistic is more extreme than a threshold determined so that the probability of type I error stays below a pre-established significance level α. Usually, if p-values are smaller than the significance level, the null hypothesis is rejected.

Type I error: probability of rejecting the null when the null is true.

Review: Hypothesis Testing

  • α: type I error — probability of rejecting H₀ when H₀ is true (false positive rate)
  • β: type II error — probability of not rejecting H₀ when H₀ is false (false negative rate)
  • Power = 1 − β

This figure shows the test statistics under H₀ and Hₐ. The decision rule is to reject H₀ if the test statistic exceeds the threshold. We choose significance level α (typically 0.05) and calculate the threshold. When running multiple tests, a much more stringent significance level is required.

P-value

P-value = P(result as extreme or more extreme than observed | H₀ is true)

Regression Approach: Single SNP

Regression Approach

\[Y = \mu + a \cdot \text{age} + \beta \cdot X + \epsilon\]

  • Parameters β estimated using MLE, least squares, etc.
  • Null hypothesis: β = 0
  • Can handle many types of traits with the same approach
  • Can correct for covariates (age, sex, ancestry)
  • Useful for prediction

Genotype: aa, aA, AA → X₁: dosage = number of A alleles

Regression Approach: Quantitative Traits

Quantitative trait (e.g., height, BMI, systolic blood pressure)

\[Y = \beta_0 + \beta_1 \cdot X_1 + \epsilon, \quad \epsilon \sim N(0, \sigma^2)\]

Linear Regression

Genotype: aa, aA, AA → X₁: dosage = number of A alleles

Quantitative traits are modeled with normal errors: mean = β₀ + β₁X₁, where X₁ is the number of minor alleles.

Regression Approach: Disease Traits

Binary trait (e.g., disease status, hypertension)

\[Y \sim \text{Bernoulli}(\pi)\]

\[\text{logit}(\pi) = \log\!\left(\frac{\pi}{1-\pi}\right) = \beta_0 + \beta_1 X_1\]

Logistic Regression — β₁ = log odds ratio

Binary traits are modeled using logistic regression. β₁ is the log odds ratio: for X₁ = 0, log odds = β₀; for X₁ = 1, log odds = β₀ + β₁.

Genome-wide Association Studies

GWAS: Overview

GWAS collects individuals (cases/controls for disease, or population samples for quantitative traits), measures genotypes at ~1 million markers, and performs association tests between the phenotype and each genetic marker (SNPs).

GWAS: Data Structure

The genotype matrix X and phenotype vector Y. Single-marker SNP association tests produce a table of results: effect size, standard error, and p-values.

GWAS: Manhattan Plot

GWAS significance threshold: 5×10⁻⁸

Ehret 2011, PMC2865585

Manhattan plots visualize GWAS results. Points above −log₁₀(5×10⁻⁸) are called GWAS significant.

QQPlot

  • A Q-Q plot is useful to present GWAS results and check for potential issues
  • X-axis: expected −log(P) under the null (uniformly distributed p-values)
  • Y-axis: observed −log(P)
  • Dots above the 45° line deserve a closer look
  • Can also detect population stratification

In a well-behaved GWAS, most points line up at the identity line, with a few departing at the upper right (true associations). Early departure from the identity line for many points suggests population structure.

GWAS SNP-Trait Discovery Timeline

GWAS have been so successful that thousands have been performed since 2005. As of 2020, more than 70,000 SNP-trait associations are reported in the GWAS Catalog.

WTCCC: First Large Scale GWAS

WTCCC

  • Wellcome Trust Case Control Consortium
  • GWAS of 2,000 cases and 3,000 shared controls for 7 diseases
  • Platform: Affymetrix 500K Set
  • Main paper published in 2007
  • Results and summaries freely available; genotype data access granted to qualified investigators

The WTCCC GWAS is a landmark study of 7 common diseases — the first large-scale GWAS performed.
